# Spotless Carwash — Under Construction

Temporary landing page for [spotlesscarwash.com](https://spotlesscarwash.com) while the main site is being rebuilt. The bays are open as usual — touchless automatic, self-serve wand bays, and Lustra Shield gloss.

Built with Create React App + Tailwind (via CDN).

## Scripts

- `npm start` — run the dev server at http://localhost:3000
- `npm run build` — production build to `build/`
- `npm test` — run tests
